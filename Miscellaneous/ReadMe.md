# PE.T Peer Tutoring Program Website

## Overview
A comprehensive, responsive website for the PE.T (Peer Tutoring Program) designed to connect university students with experienced peer tutors for technical and software development support. The website features modern web technologies, SEO optimization, interactive elements, and accessibility compliance.

## Project Timeline
- **August 5, 2025**: Project proposal approved
- **August 20, 2025**: Started POE Part 1 development
- **September 25, 2025**: Major corrections and improvements implemented
- **November 10-13, 2025**: Final enhancements and feature additions

## Features

### Core Functionality
- **Responsive Navigation Menu**: Seamless experience across mobile, tablet, and desktop devices
- **Dynamic Greetings**: Time-based greeting adjustments (morning/afternoon/evening)
- **Back to Top Button**: Smooth scroll functionality with visibility controls
- **Interactive Forms**: Contact and enquiry forms with comprehensive validation
- **Program Information**: Detailed sections on About, Services, Products, and Budget
- **Gallery Section**: Responsive image gallery showcasing tutoring sessions and events
- **Google Maps Integration**: Interactive location display with directions link
- **Smooth Scroll**: Internal navigation with smooth scrolling effects

### Advanced Features (High-Scoring Implementation)
- **SEO Optimization**: Meta titles, descriptions, and keywords on all pages
- **Gallery Lightbox**: Responsive modal lightbox for image viewing
- **Dynamic Search**: Real-time search functionality across site pages
- **Enhanced Form Feedback**: Detailed success/error messages with auto-hide
- **Interactive Maps**: Google Maps embed with directions integration

## Technologies Used
- **HTML5**: Semantic markup with accessibility features
- **CSS3**: Flexbox, Grid layouts, responsive design, animations
- **JavaScript**: DOM manipulation, event handling, form validation, lightbox functionality
- **Google Maps Embed**: Interactive location services


2. **Navigate to project directory**:
   ```bash

   ```
